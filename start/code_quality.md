Tips:
----------
如何分析代码质量:

安装Python packages:

    pip install pep8
    pip install pylint

Pep8 checke code:

    pep8 --show-source --show-pep8 a.py
    testsuite/E40.py:2:10: E401 multiple imports on one line
    import os, sys
         ^
    Imports should usually be on separate lines.

    Okay: import os\nimport sys
    E401: import sys, os
